Static corpus optimized for retrieval + citation. Prefer machine exports over HTML scraping.

Start here (for agents)

  • llms.txt — curated entrypoints
  • docs/rag-contract.md — grounding rules (“no cite = no claim”)
  • exports/manifest.json — corpus inventory + versions + hashes
  • exports/chunks.ndjson — chunk stream (NDJSON, one chunk per line)

Paper bundles

  • LocalAgent v3 — index ( exec · claims · full )
  • Brown v. Board (1954) — index ( exec · claims · full )
For grounded summaries, prefer claims.ndjson + cite chunk_ids from exports/chunks.ndjson.

RAG usage pattern

  1. Fetch manifest.json
  2. Retrieve relevant chunk_id records from chunks.ndjson
  3. Summarize with citations (chunk_ids per paragraph)
  4. If evidence is missing: output INSUFFICIENT_EVIDENCE
